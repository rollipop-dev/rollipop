---
title: SVG Support
---

SVG support is provided by the [`@rollipop/plugin-svg`](https://github.com/rollipop-dev/rollipop/tree/main/packages/plugin-svg) plugin, which transforms `.svg` files into React components using `@svgr/core`.

## Pre-requisites

To render SVGs in your application, you first need to add [react-native-svg]() native module to your application.

## Installation

```sh
yarn add -D @rollipop/plugin-svg
```

## Usage

```ts title="rollipop.config.ts"
import { svg } from '@rollipop/plugin-svg';
import { defineConfig } from 'rollipop';

export default defineConfig({
  plugins: [svg()],
});
```

The plugin's `config` hook adds `svg` to `resolve.sourceExtensions` and removes it from `resolve.assetExtensions`, so `.svg` imports are routed through the plugin's `load` hook instead of being bundled as static assets.

When the plugin is **not** added, `.svg` files are treated as regular assets (the default `resolve.assetExtensions` includes `svg`).
